## 1. 快速入门

### 1.1 系统要求

| 项目 | 要求 |
|-----|------|
| 操作系统 | Windows 7/10/11 |
| Python | 3.7 或更高版本 |
| 相机 | 支持 MVS SDK 的工业相机 |
| 运动平台 | Z 轴电动平台（串口通信） |

### 1.2 安装依赖

```bash
pip install PyQt5 numpy matplotlib pyserial
```

### 1.3 启动程序

```bash
python main.py
```

### 1.4 最简操作流程（5分钟上手）

```
1. 枚举设备 → 选择相机 → 打开设备 → 开始取流
2. 刷新串口 → 选择串口 → 连接串口
3. 菜单 → 出图 → 一键出图
4. 设置 Z 轴范围 → 点击"一键出图"
5. 完成！查看生成的合成图
```

## 2. 设备连接

### 2.1 连接相机

#### 步骤说明

| 步骤 | 操作 | 预期结果 |
|-----|------|---------|
| 1 | 点击 **"枚举设备"** | 下拉框显示可用相机列表 |
| 2 | 在下拉框中选择相机 | 显示相机名称 |
| 3 | 点击 **"打开设备"** | 状态显示"设备已打开" |
| 4 | 点击 **"开始取流"** | 实时显示相机画面 |

#### 相机参数设置（可选）

| 参数 | 建议值 | 说明 |
|-----|-------|------|
| 曝光时间 | 5000-20000 μs | 根据光照条件调整 |
| 增益 | 0-10 dB | 越高噪点越多 |

### 2.2 连接串口（Z 轴控制）

#### 步骤说明

| 步骤 | 操作 | 预期结果 |
|-----|------|---------|
| 1 | 点击 **"刷新串口"** | 下拉框显示可用串口 |
| 2 | 选择正确的串口（如 COM3） | - |
| 3 | 设置波特率（默认 115200） | - |
| 4 | 点击 **"连接串口"** | 状态显示"串口已连接" |

## 3. 一键出图

> **适用场景**：快速获得一张全焦清晰图像，最简单的操作方式

### 3.1 打开功能

```
菜单栏 → 出图(I) → 一键出图...
```

### 3.2 界面说明

```
┌─────────────────────────────────────────────────┐
│ 扫描参数（Z 轴：从上向下）                      │
│   Z 高位-起点 (mm): [2.0]     ← 扫描起始位置    │
│   Z 低位-终点 (mm): [-2.0]    ← 扫描结束位置    │
│   Z 步长 (mm):      [0.2]     ← 每步移动距离    │
│   每步延时 (s):     [0.20]    ← 稳定等待时间    │
├─────────────────────────────────────────────────┤
│ 高级选项                                        │
│   ☐ 同时生成点云数据（用于 3D 测量和导出）      │
│       Z 轴缩放系数: [1.0]                       │
│       最小锐度阈值: [20.0]                      │
├─────────────────────────────────────────────────┤
│ [进度条]                                        │
│ 状态：就绪                                      │
├─────────────────────────────────────────────────┤
│ [合成图预览区域]                                │
├─────────────────────────────────────────────────┤
│ [一键出图]  [停止]                              │
│ [另存合成图...]                                 │
│ [可视化点云] [导出点云...]  ← 勾选点云后显示    │
└─────────────────────────────────────────────────┘
```

### 3.3 基本操作步骤

#### 只要全焦图（默认）

| 步骤 | 操作 | 说明 |
|-----|------|------|
| 1 | 设置 Z 高位 | 扫描起始位置（上方） |
| 2 | 设置 Z 低位 | 扫描结束位置（下方） |
| 3 | 设置 Z 步长 | 建议 0.1-0.3 mm |
| 4 | 设置每步延时 | 建议 0.15-0.25 s |
| 5 | 点击 **"一键出图"** | 开始扫描 |
| 6 | 等待完成 | 进度条显示进度 |
| 7 | 查看预览 | 合成图自动显示 |

#### 同时生成点云

| 步骤 | 操作 | 说明 |
|-----|------|------|
| 1 | 勾选 **"同时生成点云数据"** | 显示点云参数 |
| 2 | 设置 Z 轴缩放系数 | 默认 1.0 |
| 3 | 设置最小锐度阈值 | 默认 20.0，越大过滤越严格 |
| 4 | 点击 **"一键出图"** | 开始扫描 |
| 5 | 完成后点击 **"可视化点云"** | 查看 3D 效果 |
| 6 | 点击 **"导出点云..."** | 保存 PLY/CSV 文件 |

### 3.4 输出文件

| 文件类型 | 格式 | 自动保存 | 说明 |
|---------|------|---------|------|
| 全焦合成图 | BMP | ✅ 是 | composite_YYYYMMDD_HHMMSS.bmp |
| 点云数据 | PLY/CSV | ❌ 手动 | 需点击"导出点云"保存 |

> **适用场景**：需要完整的三维测量数据，支持双向扫描

```
菜单栏 → 三维重建(3D) → ① 点云重建...
```

### 4.2 界面说明

```
┌─────────────────────────────────────────┐
│ 扫描参数（Z 轴）                        │
│   Z 起始位置 (mm): [-2.0]               │
│   Z 结束位置 (mm): [2.0]                │
│   Z 步长 (mm):     [0.1]                │
│   每步延时 (s):    [0.15]               │
├─────────────────────────────────────────┤
│ 点云参数                                │
│   Z 轴缩放系数:    [1.0]                │
│   最小锐度阈值:    [20.0]               │
├─────────────────────────────────────────┤
│ [进度条]                                │
│ 状态：就绪                              │
├─────────────────────────────────────────┤
│ [开始重建]  [停止]                      │
│ [可视化点云]  [导出点云]                │
└─────────────────────────────────────────┘
```

### 4.3 操作步骤

| 步骤 | 操作 | 说明 |
|-----|------|------|
| 1 | 设置 Z 起始位置 | 扫描下限 |
| 2 | 设置 Z 结束位置 | 扫描上限 |
| 3 | 设置 Z 步长 | 建议 0.05-0.15 mm |
| 4 | 设置每步延时 | 建议 0.15-0.25 s |
| 5 | 设置 Z 轴缩放系数 | 调整深度比例 |
| 6 | 设置最小锐度阈值 | 过滤模糊区域 |
| 7 | 点击 **"开始重建"** | 开始扫描 |
| 8 | 等待完成 | 显示点数和覆盖率 |
| 9 | 点击 **"可视化点云"** | 查看 3D 效果 |
| 10 | 点击 **"导出点云"** | 保存文件 |

### 4.4 与一键出图的区别

| 特性 | 点云重建 | 一键出图 |
|-----|---------|---------|
| 扫描方向 | 双向（起始→结束） | 单向（高→低） |
| 点云生成 | 必选 | 可选 |
| 参数灵活性 | 高 | 中 |
| 操作复杂度 | 中 | 简单 |
| 适用场景 | 精密测量 | 快速出图 |

## 5. 连续扫描重建

> **适用场景**：高速扫描，支持嵌套精扫提高精度

### 5.1 打开功能

```
菜单栏 → 三维重建(3D) → ② 连续扫描重建...
```

### 5.2 操作步骤

| 步骤 | 操作 | 说明 |
|-----|------|------|
| 1 | 设置 Z 轴范围 | 起始和结束位置 |
| 2 | 设置扫描速度 | mm/s |
| 3 | 设置采帧间隔 | 秒 |
| 4 | （可选）勾选嵌套精扫 | 提高精度 |
| 5 | 点击 **"开始扫描"** | 开始 |
| 6 | 查看结果并导出 | - |

### 5.3 嵌套精扫说明

```
第一轮：粗扫（快速确定焦点范围）
   ↓
分析：找到最清晰的 Z 范围
   ↓
第二轮：精扫（在该范围内细扫）
   ↓
融合：合并粗扫和精扫结果
```

## 6. 参数说明

### 6.1 扫描参数

| 参数 | 单位 | 建议范围 | 影响 |
|-----|------|---------|------|
| Z 步长 | mm | 0.05-0.3 | 越小越精确，但越慢 |
| 每步延时 | s | 0.1-0.5 | 确保平台稳定 |
| 扫描速度 | mm/s | 0.5-2.0 | （连续扫描）越快越快 |
| 采帧间隔 | s | 0.05-0.2 | （连续扫描）越小帧数越多 |

### 6.2 点云参数

| 参数 | 默认值 | 说明 |
|-----|-------|------|
| Z 轴缩放系数 | 1.0 | 调整深度方向的比例 |
| 最小锐度阈值 | 20.0 | 低于此值的像素被过滤，建议 10-100 |

### 6.3 参数调整建议

#### 场景 1：扫描速度优先

```
Z 步长: 0.2-0.3 mm
每步延时: 0.1-0.15 s
最小锐度阈值: 10-15
```

#### 场景 2：精度优先

```
Z 步长: 0.05-0.1 mm
每步延时: 0.2-0.3 s
最小锐度阈值: 30-50
```

#### 场景 3：点云质量差

```
↓ 减小 Z 步长（0.05 mm）
↓ 增加每步延时（0.25 s）
↓ 降低最小锐度阈值（10-15）
↓ 检查相机曝光是否合适
```

## 7. 输出文件

### 7.1 文件类型

| 类型 | 格式 | 内容 | 用途 |
|-----|------|------|------|
| 全焦合成图 | BMP/PNG | 灰度图像 | 文档记录、打印 |
| 深度图 | PNG | 灰度图，亮=高 | 深度分析 |
| 点云数据 | PLY | X,Y,Z,强度 | 3D 软件查看 |
| 点云数据 | CSV | 表格数据 | Excel 分析 |

### 7.2 文件命名规则

```
composite_YYYYMMDD_HHMMSS.bmp    # 合成图
pointcloud_YYYYMMDD_HHMMSS.ply  # 点云
pointcloud_YYYYMMDD_HHMMSS.csv  # 数据
```

### 7.3 点云文件查看

#### 推荐软件

| 软件 | 平台 | 特点 |
|-----|------|------|
| MeshLab | 跨平台 | 免费开源，功能强大 |
| CloudCompare | 跨平台 | 免费开源，专业点云处理 |
| Blender | 跨平台 | 免费开源，支持渲染 |

#### MeshLab 打开步骤

```
1. 打开 MeshLab
2. File → Import Mesh
3. 选择 .ply 文件
4. 使用鼠标旋转查看
```

### 7.4 CSV 文件格式

## 8. 故障排除

### 8.1 相机问题

| 症状 | 可能原因 | 解决方法 |
|-----|---------|---------|
| 枚举不到设备 | 相机未连接/未供电 | 检查 USB/电源连接 |
| 打开设备失败 | SDK 未安装 | 安装相机厂商 SDK |
| 画面全黑 | 曝光时间过短 | 增加曝光时间 |
| 画面全白 | 曝光时间过长/光线过强 | 减少曝光或遮光 |
| 画面模糊 | 焦点不对/震动 | 调整焦距，稳定平台 |

### 8.2 串口问题

| 症状 | 可能原因 | 解决方法 |
|-----|---------|---------|
| 刷新不到串口 | 驱动未安装/未连接 | 安装 CH340/CP2102 驱动 |
| 连接失败 | 波特率错误 | 尝试 9600/115200 |
| Z 轴不动 | 串口选择错误 | 尝试其他串口 |
| Z 轴运动异常 | 平台未使能 | 检查平台开关/限位 |

### 8.3 扫描问题

| 症状 | 可能原因 | 解决方法 |
|-----|---------|---------|
| 扫描中断 | 超时/丢帧 | 增加延时，检查连接 |
| 合成图模糊 | 步长过大 | 减小 Z 步长 |
| 点云空洞多 | 锐度阈值过高 | 降低最小锐度阈值 |
| 点云噪点多 | 锐度阈值过低 | 提高最小锐度阈值 |
| 深度不准确 | 扫描范围设置错误 | 调整 Z 轴范围 |

### 8.4 性能问题

| 症状 | 可能原因 | 解决方法 |
|-----|---------|---------|
| 扫描很慢 | 步长太小/延时太长 | 增大步长，减少延时 |
| 可视化卡顿 | 点数太多 | 软件自动采样（限制 6 万点） |
| 内存不足 | 帧数太多/分辨率太高 | 减小扫描范围或降低相机分辨率 |

## 附录 A：快捷操作参考
