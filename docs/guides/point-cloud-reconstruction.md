# 三维点云高质量采集与预览

## 重建能力

“点云重建”采用单相机 Z-stack 的 Depth from Focus（DFF）方法，输出可见上表面的 2.5D 高度场。升级后的实时流程包含：

- Tenengrad 与 Laplacian 组合焦点评价；
- 相邻焦层平移配准；
- 最佳/次佳峰及峰值相邻层的流式统计；
- 三点抛物线亚步距深度拟合；
- 焦点峰值显著性和曲率置信度；
- 低置信度修复、异常深度抑制和边缘保持平滑；
- 黑底、高度伪彩、规则网格降采样的交互式点云预览。

该方法不能恢复背面、遮挡面或垂直悬垂结构，也不能从透明物体的内部纹理中可靠推断真实外表面。

## 推荐采集流程

1. 固定相机曝光、增益、白平衡和光源亮度，采集期间不要自动变化。
2. 先用 `0.05 mm` 步距粗扫，确认物体所有区域的清晰焦层都位于扫描范围内部。
3. 缩小 Z 范围后使用 `0.01–0.02 mm` 步距精扫。步距越小，高度台阶越弱，但帧数和采集时间越大。
4. 高反光样品降低曝光并使用漫反射照明；条件允许时使用交叉偏振片抑制镜面高光。
5. 让物体与背景具有明显亮度差。黑色或白色无纹理背景更利于自动生成表面掩码。
6. 点击“可视化点云”后可用鼠标旋转和缩放。颜色由蓝到红表示相对高度由低到高。

## 参数说明

- `Z 轴缩放系数`：改变导出点云的 Z 坐标，也改变预览高度。计量用途保持为 `1.0`；仅展示时可适当增大。
- `最小锐度`：`0–100` 表示相对于全图高锐度参考值的百分比。默认 `5` 会保留较多表面；背景噪点较多时可以逐步增大。
- 高质量结果优先通过减小 Z 步距、改善照明和扩大正确扫描范围获得，不建议只依赖增大平滑强度。

## 输出与质量诊断

每次扫描会独立生成一个 `时间_recon` 文件夹，其中包含：

- `full_focus.png`：全焦融合图；
- `depth_um16.tif`：亚步距拟合前的相对高度；
- `surface_depth_um16.tif`：置信度修复后的表面高度；
- `focus_confidence.png`：焦点深度置信度；
- `surface_mask.png`：点云表面掩码；
- `point_cloud.ply`：毫米坐标点云；
- `frames/`：采集时同步保存的原始 Z-stack；
- `manifest.json`：参数、配准偏移和质量指标。

界面会自动提示两类常见问题：

- `边界焦层`比例过高：最佳焦点大量落在第一层或最后一层，应扩大或平移 Z 扫描范围；
- `过曝`比例过高：应降低曝光或改善照明，否则焦点评价可能追踪高光而不是真实表面。

对于精密测量，应使用已知台阶高度的标准件验证高度误差、重复性和 XY/Z 标定，不应仅凭彩色预览判断精度。
